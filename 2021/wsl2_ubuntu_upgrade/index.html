WSL2 Ubuntu distro upgrade

Written by yk6 with ♥ on October 9, 2021 in

Upgrade WSL2 Ubuntu OS

  1. Upgrade all packages to latest version
sudo apt-get update && sudo apt-get upgrade
  1. Install update manager
sudo apt-get install update-manager-core
  1. Set Prompt value in /etc/update-manager/release-upgrades to select between lts or normal release
  2. Remove snapd by the follow cmd, (solution source)
sudo apt remove snapd
  1. Perform the upgrade, add -d option for getting the latest experiment release
sudo do-release-upgrade
  1. After completing, confirm the release version
lsb_release -a
