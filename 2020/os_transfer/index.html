How to transfer OS from 1 computer to another

  1. Prepare empty HDD/SSD
  2. Download https://tw.easeus.com/backup-recovery/migrate-windows-10-to-ssd.html
  3. Follow instruction in step 2
  4. HDD/SSD must be bigger than actual physical size of disk that is being replicated
    • eg. ur physical disk is splited into C drive(512G) and D drive(512G)
    • then a HDD/SSD of 1T is required even though C and D drive is virtually different
    • or you can delete D drive (delete volumn/partition) via disk manager in Windows
    • giving a 512G of C drive and unallocated 512G of nothing
  5. Perform actual copying action
  6. Boot with new disk and check window is working fine
  7. If you want to reset Windows, check that all user content you wish to preserve is copied out
    • else, after resetting, new user may not access the folders as it have no permission granted
    • as the the permission still belongs to the original user(the one before you reset)
  8. If step 7 is done but files are not accessible, read up on

https://answers.microsoft.com/zh-hant/windows/forum/all/%E6%B2%92%E6%9C%89%E6%AC%8A%E9%99%90%E5%8F%96/479befbc-5a15-4a7c-9e7d-55baeb3e44b6
